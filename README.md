# 📊 Data Analytics Project

## 📌 Overview

This project demonstrates an end-to-end **data analytics workflow** using Python, SQL, and Power BI. The objective is to analyze a dataset, extract meaningful insights, and present them through an interactive dashboard and presentation.

The project includes **data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, dashboard creation, and reporting**. It shows practical skills required for **Data Analyst roles**.

---

## 📂 Dataset

The dataset used in this project contains structured data that was analyzed to identify patterns, trends, and insights.

**Key characteristics:**

* Structured tabular dataset
* Multiple features/columns
* Contains missing values and inconsistencies that required cleaning
* Suitable for SQL queries and visualization

*(customer_shopping_behavior.csv)*

## 🔎 Project Workflow / Steps

### 1️⃣ Data Loading

* Imported dataset using **Python (Pandas)**
* Inspected structure and column types
* Checked dataset size and initial records

### 2️⃣ Exploratory Data Analysis (EDA)

* Analyzed data distributions
* Identified trends and relationships
* Generated visualizations to understand patterns

### 3️⃣ Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized column formats
* Corrected inconsistent values

### 4️⃣ SQL Data Analysis

Data was loaded into **SQL database** for further analysis.

Example tasks:

* Aggregation queries
* Filtering and grouping
* Joins and subqueries
* Trend analysis

SQL queries were executed using:

* **PostgreSQL**
* **MySQL**
* **SQL Server**

### 5️⃣ Dashboard Creation

An interactive **Power BI dashboard** was created to visualize key insights.

Dashboard includes:

* KPI metrics
* Trend analysis
* Category breakdown
* Interactive filters and charts

### 6️⃣ Reporting & Presentation

* Key findings were documented in a **data analysis report**
* A professional **PowerPoint presentation** was created using **Gamma** to communicate insights clearly.

## ▶️ How to Run the Project

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/data-analytics-project.git
cd data-analytics-project
```

### 2️⃣ Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### 3️⃣ Run Python Analysis

Open the Jupyter Notebook and run:

```bash
jupyter notebook
```

### 4️⃣ Run SQL Queries

Import the dataset into your database:

* PostgreSQL
* MySQL
* SQL Server

Execute the SQL scripts included in the repository.

### 5️⃣ Open Power BI Dashboard

Open the `.pbix` file in **Power BI Desktop** to explore the interactive dashboard.
